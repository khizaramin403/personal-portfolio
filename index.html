Facial Recognition Attendance System

A smart attendance system that uses facial recognition technology combined with geo-restriction to accurately mark employee attendance — only within the office premises. Eliminating manual processes and proxy attendance completely.

Key Features:
  • • Facial Recognition: Automatically identifies and verifies employees via face scan.
  • • Geo Restriction: Attendance only marks within defined office location boundaries.
  • • Proxy Prevention: Eliminates buddy punching and manual attendance fraud.
  • • Real-Time Tracking: Instant attendance logs for HR and management.
  • • Employee Management: Add, update, and manage employee face data easily.
Tech Stack:

Flutter, Supabase, Node.js, ML Kit (Face Detection)
